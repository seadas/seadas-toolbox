l1aextract_modis

Creates an extracted L1A MODIS file.

Location

Location

SeaDAS-Toolobox -> SeaDAS Processors -> extractor

Details

l1aextract_modis extracts a MODIS Level 1A sub-scene from an input Aqua or Terra L1A and GEO file. Users can specify a sub-scene either using longitude/latitude of southwest and northeast corners (swlon, swlat, nelon, nelat) or using start/end pixel numbers in conjunction with start/end lines. The former is referred to as lon/lat arguments and the latter is referred as pixel arguments. Once a user choose to specify the sub-scene using the lon/lat option, all four arguments must be specified. The program will fail to extract if any argument is missing.

Alternatively, users can directly specify pixel arguments to extract a sub-scene.

Command Line Usage

 
This is version 0.76 of l1aextract_modis (compiled on Nov 17 2022 15:34:32)

Usage: l1aextract_modis infile spixl epixl sline eline outfile
   where:
	infile   - input MODIS L1A datafile
	spixl    - start pixel number (1-based)
	epixl    - end pixel number (1-based)
	sline    - start line (1-based)
	eline    - end line (1-based)
	outfile  - output file name

Note: Enter line number NOT scan number!
