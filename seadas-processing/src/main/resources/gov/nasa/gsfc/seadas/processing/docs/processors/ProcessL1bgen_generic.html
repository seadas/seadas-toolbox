l1bgen_generic

l1bgen_generic

Creates a Level 1B file from an input Level 1A file.

Location
Details
Command Line Usage

Location

SeaDAS-Toolobox -> SeaDAS Processors -> l1bgen_generic

Details

l1bgen_generic reads in a SeaWiFS Level-1A file (HRPT, LAC, or GAC) and generates a L1B file by applying the sensor calibration at the same resolution and location as the Level-1A data.
