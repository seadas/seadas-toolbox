About SeaDAS 9.2.0

Introduction

The SeaDAS software is a set of executable tools and an application programming interface (API) facilitating the use of the data products of a number of earth observation satellites. SeaDAS is based on the ESA SNAP software. SNAP has initially been developed to support the data of the MERIS, AATSR and ASAR sensors onboard the ESA Envisat platform. SNAP is developed under contract to ESA/ESRIN.

Directory Structure

This is the structure of the SeaDAS installation directory:

  • bin - SeaDAS executable and SeaDAS command-line tools.
  • config - SeaDAS configuration file.
  • lib - Common libraries used by SeaDAS.
  • modules - Installed SeaDAS modules and plugins.
  • licenses - 3rd party licenses and EULAs of libraries.

If SeaDAS is started the first time, a .seadas directory is created in the user's home directory. SeaDAS uses this directory to store user preferences, auxiliary and temporary data.

Third Party Content

SeaDAS includes some items that have been sourced from third parties. The licenses for these items are provided in the licenses directory (see above).
