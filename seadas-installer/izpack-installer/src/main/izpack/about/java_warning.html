SeaDAS requires Java 21 (JDK), not just a JRE.

Detected Java version: ${JAVA_VERSION}

SeaDAS 10 needs a full Java Development Kit (JDK) 21 or newer. A plain Java Runtime Environment (JRE) is not sufficient.

If you already have a JDK 21 installed on your system, please enter its path in the field below or use the Browse… button to locate it.

If you don’t yet have Java 21, you can:

  • Download and install Temurin 21 (or OpenJDK 21) from adoptium.net, or
  • Use the SeaDAS installer with bundled Java available at SeaDAS Downloads Page.

Tip: On most systems, the JDK directory contains a bin folder with java and javac inside. Example paths:
• /usr/lib/jvm/jdk-21 (Linux)
• C:\Program Files\Eclipse Adoptium\jdk-21 (Windows)
• /Library/Java/JavaVirtualMachines/jdk-21.jdk/Contents/Home (macOS)

Once a valid JDK 21 path is detected, the installation will continue automatically.
